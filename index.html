Você pode ter seu próprio servidor de arquivos

Um servidor que esteja onde você saiba, que apenas você controla o conteúdo, com acesso único ou com usuários que você cadastrar, de tamanho físico menor que um codificador de TV a cabo e consumo menor que 60W.h.

Hardware simples e robusto
  • Usa Raspberry Pi 3B Quadcore 1GB RAM
  • Armazenamento de até 4TB via HDD/SSD (SupTronix X820)
  • Placa de potência P4/USB-C dedicada (SupTronix X735)
  • Tela de 0,96in OLED 128x64px para monitoramento
  • Case de PLA em impressão 3D FDM, biodegradável
  • Ventilação própria com controle PWM com exaustão

Software escalável e privado
  • SO Linux Raspbian baseado em Debian, server ou desktop
  • Privacidade total, acessos controlados por você
  • Acesso via VPN de sua propriedade, sem servidores externos
  • Compartilhamento via SMB, simples e robusto
  • Acesso via WI-FI 2.4GHz ou Ethernet/100
  • Crie os drivers virtuais precisar, com acessos independentes
